# Security Policy

Clauchy reads Claude Code credentials and transcripts and modifies Waybar and
Hyprland configuration during installation. Please report security problems
privately rather than opening a public issue.

## Report A Vulnerability

Use GitHub's private vulnerability reporting page:

https://github.com/jesusrobot0/clauchy/security/advisories/new

Include the affected version, reproduction steps, expected impact, and any
suggested mitigation. Do not include real access tokens, refresh tokens, or
private transcript content.

Supported versions are the latest tagged release and the current `main` branch.
